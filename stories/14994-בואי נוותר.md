# בואי נוותר

בואי נוותר.

בואי ונזנח את הרצון הנורא.

הזמן הריק והתמים איננו,

אינך רואה?

היקום הנוגה והסתמי הזה

הוא רק תרמית נוראה.



ואת, מי את בחיי?

את, נמצאת בתיאוריי.

את, האם את התרופה,

או מגיפה הזורמת בין תאיי?



בואי נברח.

בואי ונחיה ביקום אחר.

נביחות ריקניות ירחפו בינינו,

ואת הקסם נמחזר.

החיים הקצרים וחסרי המהות

ייזנחו ללא רחם.



ואת, מי את בחיי?

את, נמצאת בתיאוריי.

את, האם את התרופה,

או מגיפה הזורמת בין תאיי?



בואי נסיים.

בואי ונרחץ ברוח החג.

התיקון יבוא ויחתום את חיינו,

ואת קרבות הסרק.

הסביבה החיוורת והקפואה

יוצרת בי מחנק.



ואת, האם את בחיי?

את, נמצאת בתיאוריי.

את, את רק המגיפה

הזורמת בין תאיי.



בואי נוותר.

בואי ונזנח את הרצון הנורא.

הזמן הריק והתמים איננו,

אינך רואה?
