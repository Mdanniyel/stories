# חמשת החושים

שומעת את קולך מתנגן באוזני כמנגינה נעימה שלא אשכח לעולם...

מקשיבה למילים שלך... מילים שמבטאות את אהבתך הרבה אלי. מילים שמרעידות אותי כל פעם מחדש..

מאזינה לנשימות שלך...כאשר הן רגועות וכאשר הן סוערות וכמהות אלי...


מביטה בעינייך היפות שמספרות לי את סיפור אהבתנו.

מסתכלת על קימורי גופך העירום... את לא יודעת כמה שאת יפה...

רואה את פנייך מלאות התשוקה כשאת רוצה שאני אגע...


מריחה אותך... את כל כולך... ריח שמיוחד רק לך.

ריח שמשאיר לי ניחוח נעים על הכרית... 

מריחה ריח של אהבה... של תשוקה... של בערה עצומה...


טועמת את שפתייך הרכות... את לשונך החמה...

טועמת את צווארך המתוח...

את שדייך היפים... ופטמותייך הזקורות...


נוגעת בך... מלטפת את פנייך, את שפתייך.

מלטפת את שדייך... מרגישה אותך תחתי...

מעבירה את ידי בין רגלייך ונוגעת בך.

מרגישה אותך מפנים, את החום העצום שלך...


שומעת את אנחותייך,

מביטה בפנייך,

מריחה ניחוח משכר,

טועמת את שפתייך,

ומלטפת אותך מבפנים...


את כל כך יפה,

הזיכרון הזה יישאר תמיד איתי.
