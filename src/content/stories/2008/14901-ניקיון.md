# ניקיון

תבואי היום,

בסוף היום,

ותנקי אותי,

תסבני את גופי הקר,

תרחצי אותי במים,

תסובבי את הפגיון,

תראי איך היופי ניגר,

תניחי אותי באמבטיית קרח.

תסירי את הבגדים מגופך

הנקי,

תריחי את האוויר

ותייבבי

ותאהבי

ותקווי.

הקרח יקרר את גופתי

ובינתיים,

תוציאי מהארון חומר ניקוי אורגני,

היוצר סביבה נקייה,

רקובה,

תוסיפי אותו פנימה

ותתמזגי בניקיון

ובי.
