# הריח שלך

אני אוהבת את הריח שלך כשאת רק פוקחת עיניים בבוקר כולך עדיין ישנונית,
אני אוהבת את הריח שלך כשאת מוכנה לצאת לעבודה, ליום חדש,
אני אוהבת הריח שלך כשאת מתאמנת במרץ,
אני אוהבת את הריח שלך במקלחת כשבועות סבון מלטפות את גופך,
אני אוהבת את הריח שלך כשאנחנו עושות אהבה, סקס סוער,
אני אוהבת את הריח שלך כשאת מזיעה עלי רטובה כולך מכל איבר אפשרי בגופך,
אני אוהבת את הריח שלך כשאנחנו מכורבלות אחת בשנייה בין הסדינים,
אני אוהבת את הריח, שלך.
